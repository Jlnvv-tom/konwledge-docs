#### 8个偏差详细内容

**1. 基本归因错误（Fundamental Attribution Error）**
解释：用他人的人品和性格来评判他人，却用外部情况来为自己开脱。
案例：你迟到了，说"堵车了"；别人迟到了，你想"他就是不守时的人"。
机制：当观察他人时，关注的是行为本身；当我们自己犯错时，关注的是情境压力。

**2. 光晕效应（Halo Effect）**
解释：如果一个人有某个正面特质，这种正面印象会蔓延到其整体，反之亦然。
案例：长相好看的人被认为更聪明、更有能力；穿着邋遢的人被认为不可靠。
机制：认知经济性——大脑用单一特征推断整体特征，以节省认知资源。

**3. 可得性启发法（Availability Heuristic）**
解释：做决定时，依赖于脑海中直接浮现的例子，而不是真实统计数据。
案例：飞机失事新闻后，大量人取消航班改乘汽车，尽管飞机实际上更安全。
机制：越是生动、情绪化、近期的事件，越容易从记忆中检索，从而被高估其发生概率。

**4. 谷歌效应（Google Effect / Digital Amnesia）**
解释：倾向于遗忘那些容易通过搜索引擎搜索到的信息。
案例：你记不住某个明星的出生年份，因为知道"随时可以搜到"。
机制：外部存储（搜索引擎）减少了大脑编码和存储信息的必要性。

**5. 蔡加尼克效应（Zeigarnik Effect）**
解释：相比于已完成的工作，人们更容易记得未完成的工作。
案例：电影在悬念处结束，你一整晚都在想结局；完结的剧反而没那么牵挂。
机制：未完成的任务会在认知系统中保持"激活状态"，直到被关闭。

**6. 宜家效应（IKEA Effect）**
解释：我们会高估自己参与制作的事物或自己组装的东西的价值。
案例：自己组装的家具觉得特别好，哪怕其实有点歪；自己做的菜觉得格外好吃。
机制：参与创造产生了禀赋效应和能力幻觉——我们喜欢自己亲手创造的东西，并将其价值与自我价值绑定。

**7. 虚假记忆（False Memory）**
解释：我们会错把想象当做真实的记忆。
案例：在引导性提问下，受试者会"记得"从未发生的事件（如童年差点走失）。
机制：记忆不是录像回放，而是重建过程；想象的内容在大脑中产生的神经活动与真实经历高度相似。

**8. 隐记忆（Implicit Memory）**
解释：我们会错把真实的记忆当做想象。
案例：某个场景似曾相识（Céline Dion的"Déjà Vu"体验），实际上是你以前确实经历过但无法有意识地提取。
机制：无意识的记忆提取——记忆存在但有意识地无法访问，被误认为是"凭空想象"。

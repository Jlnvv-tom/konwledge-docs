# 认知偏差手册：50种认知性偏差深度解读

> **作者：** 怕浪猫  
> **适合人群：** 希望提升决策质量、深入理解人性、提升认知水平的学习者

大脑每天都在对我们撒谎，而我们自己毫不知情。

从"这个股票跌了这么多该涨了"到"大家都这么说肯定没错"，从"我肯定不会受影响"到"这个分析太准了简直就是我"——这些看似正常的想法背后，隐藏着50个已经被心理学研究验证的认知偏差。

本手册将50个认知偏差分为6大类别，通过真实案例、心理学研究和实用策略，帮你系统性地理解大脑的决策陷阱，并在关键时刻识别并规避它们。

3. **对照表使用**：每个章节末尾的对照表，建议收藏便于日常查阅

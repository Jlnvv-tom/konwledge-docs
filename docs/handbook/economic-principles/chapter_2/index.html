第2章 市场如何运行——供给与需求

2.1 需求定律：为什么涨价，买的人就少了​

2.1.1 需求量与需求的区别​

需求量指的是在某个特定价格下，消费者愿意并且能够购买的数量，它对应的是需求曲线上的一个点。而需求，指的是在每一个可能的价格下，消费者愿意购买的数量总和，它对应的是整条曲线本身。需求曲线的位置由多种因素共同决定，而需求量只是这条曲线上随着价格变动而移动的一个点。

用一个生活中的例子把这两个概念彻底说清楚。假设苹果的价格是每斤五元，你每周买三斤。如果超市打折，苹果降到每斤两元，你发现买起来不心疼了，于是每周买八斤。这是价格变化引起的需求量变化，沿同一条需求曲线的滑动——价格变，需求量跟着变，但曲线本身没动。但如果媒体突然大篇幅报道"每天吃一个苹果可以显著降低心脏病风险"，全民健康意识大幅提升。这时候，即使苹果的价格没有变，你愿意购买的数量也会增加——这不再是沿曲线移动了，而是整条需求曲线向右平移。向右移动意味着需求增加，向左移动意味着需求减少。

2.1.2 影响需求曲线位置的五大因素​

影响需求曲线位置的因素有以下五个核心变量：收入水平、相关商品价格、消费者偏好、消费者对未来的预期，以及消费者人数。

收入增加了，人们在大多数商品上愿意购买的数量都会增加，需求曲线右移。如果经济衰退，收入下降，需求曲线左移。相关商品价格的影响比较微妙：如果 A 商品涨价导致 B 商品的需求量增加，A 和 B 就是替代品关系，比如可口可乐和百事可乐；如果 A 涨价导致 B 的需求量也减少，A 和 B 就是互补品关系，比如汽油和汽车。消费者偏好的变化往往是潜移默化的，有时候一场疫情、一段社交媒体的爆款视频、一本畅销书，都能在短时间内让某类商品的需求曲线发生显著位移。消费者对未来的预期也很有意思：如果人们预期未来房价会大涨，购房需求曲线可能立即右移——大家都想趁低价赶紧买。

2.1.3 ceteris paribus：其他条件不变的假设​

需求定律还有一个重要的隐藏假设：其他条件不变。英文里叫 ceteris paribus，拉丁语，意思是"其他条件相同"。我们在第一章里已经接触过这个概念，现在它在需求定律里再次出现了——价格和需求量之间的反向关系，是在偏好、收入、其他商品价格等因素保持不变的前提下才成立的。

下次有人跟你说"油价涨了，加油的人不是照样排队"的时候，可以告诉他：单独看油价这个变量，价格涨了，需求量确实会下降，这是毫无疑问的经济规律；如果排队的人反而更多了，那一定是其他某个条件发生了变化，比如正值国庆长假大家都在自驾出行，或者人们预期油价还会继续涨所以提前来加油。价格之外的因素变化，会通过曲线移动来体现，而不是否定价格与需求量之间的反向关系。

2.2 供给定律：为什么东西贵了，商家就愿意多卖​

讲解完买家的行为逻辑，现在换到卖家这边。供给定律的内容和需求定律恰好对称：价格上升，供给量增加；价格下降，供给量减少。两者之间是一种正向的、规律性的关系，经济学称之为供给定理。这个定律的核心含义是：价格是对生产者行为的激励。价格越高，生产者扩大生产规模、投入更多资源的动力就越强；价格越低，这种动力就越弱。

如果苹果的批发价从每斤三元涨到了每斤八元，果农会想方设法多种一些、多收一些，把货更多地投放到市场上去。因为在更高的价格下，种苹果的利润空间更大，覆盖成本之后还有可观的盈余，他甚至可能把原本种梨的土地改种苹果，多雇几个人手来采摘，多买一些化肥和农药加大投入。相反，如果苹果价格跌到五毛一斤，刨去租地费用、种植成本、人力成本之后几乎不赚钱甚至亏本，果农就会减少种植，把精力转向其他利润更高的作物，或者干脆让一些苹果烂在树上——因为采摘和运输的成本已经超过了销售收益。

2.2.1 供给量与供给的区别​

和需求部分一样，供给这边也要严格区分"供给量"和"供给"这两个层次。供给量是在某个特定价格下生产者愿意提供的数量，对应供给曲线上的一个点。供给是不同价格下供给量的整体表现，对应一条完整的曲线。

如果种植苹果的肥料价格突然大幅下跌，或者出现了新的高产、抗病苹果品种，果农在每一个价格下都愿意提供更多苹果，这时候供给曲线就会向右平移——注意，这里说的是供给增加，而不是供给量增加。如果苹果价格本身上涨了，果农卖得更多了，那只是沿供给曲线向上移动，曲线本身并没有动，变化的只是同一个供给意愿在更高价格下的不同数量表现。

2.2.2 影响供给曲线位置的因素​

影响供给曲线位置的因素有以下五个核心变量：生产成本、技术水平、相关商品价格、生产者对未来的预期，以及市场上的生产者数量。

生产成本上升，供给曲线向左移动——因为成本高了，同样的价格下生产者赚得更少。技术进步让产量大幅提高，同样的价格下能提供更多商品，曲线向右移动。相关商品价格的逻辑是：如果种植苹果和种植梨之间存在土地和劳动力的竞争关系，梨的价格上涨会让更多农民去种梨，从而减少苹果的供给，苹果的供给曲线向左移动。生产者对未来的预期则比较直观：如果农民预期未来苹果价格会大涨，他们可能会把一部分今天的存货囤起来，减少今天的供给，同时扩大种植规模等待未来卖更好的价钱。

2.3 均衡价格：看不见的手如何把价格拉到"刚刚好"​

现在把需求曲线和供给曲线放到同一张图里。需求曲线向右下方倾斜，供给曲线向右上方倾斜，在二维坐标系里它们必然会相交于某一点。这个交点代表的价格，就是均衡价格；这个交点代表的交易数量，就是均衡数量。顾名思义，均衡就是供需平衡的状态——在这个价格下，买家愿意买的数量恰好等于卖家愿意卖的数量。

市场从不宣告自己的规则，但它永远在向平衡收敛。

用文字示意图来描述这张供需交叉图：假设坐标系纵轴是价格 P，横轴是数量 Q。

  P (价格)
  ↑
  |            S（供给曲线，向右上倾斜）
  |           /
  |          /
  |         /  均衡点 E
  |        /  ● (P*, Q*)
  |       /  /
  |      / /
  |     / /
  |    / /
  |   / /
  |  / /
  | / /
  |//
  +———————————————→ Q (数量)

在这个均衡点，没有过剩也没有短缺。市场价格低于均衡价格时，需求量大于供给量，形成短缺，价格会被推高；市场价格高于均衡价格时，供给量大于需求量，形成过剩，价格会被压低。市场正是在这种自发的价格调整中不断趋向均衡的。

理解均衡的稳定性很重要：市场并不需要一个中央计划者来告诉价格应该定在哪里，价格本身就像一只看不见的手，自动引导供需趋向平衡。这不是说均衡是一成不变的——需求或供给的任何一条曲线发生位移，均衡点就会移动到一个新的位置。

2.4 供需变动：曲线移动与沿曲线移动，区别到底在哪里​

这是整个供需框架中最容易混淆的地方，也是考试和实际分析中最容易出错的地方，必须单独拆开来讲。

沿曲线移动：价格变化导致需求量或供给量沿同一条固定曲线移动。这是价格自身变化引起的，数量变化对应的是曲线上点的移动。

曲线移动：需求曲线或供给曲线本身发生了位移，在每一个价格水平上数量都发生了变化。这是价格以外的因素（收入、偏好、技术、成本等）变化引起的，整条曲线发生了平行位移。

用一个具体案例来说明两者的区别。2020年初，新冠疫情突然爆发，口罩的需求发生了什么变化？价格并没有变——口罩当时的市价还没有来得及调整——但每一个价格水平下，人们愿意购买的口罩数量都大幅增加了：这就是需求曲线向右平移，而不是沿曲线移动。后来口罩价格大涨，这是需求曲线右移之后，新的均衡点沿着供给曲线向上移动到达的结果。

再看一个相反方向的例子。2021年，全球芯片短缺导致汽车产量下降。芯片短缺意味着生产成本上升，供给曲线向左移动。即使市场对汽车的需求没有变化，汽车的均衡价格也会上涨——这是供给侧收缩带来的价格上升，而不是需求增加引起的。

所以，当你在新闻里看到某个商品价格上涨时，第一反应应该是追问：这是需求曲线移动了，还是供给曲线移动了，还是两条曲线同时移动了？答案不同，政策含义和市场预期就完全不同。

2.5 价格管制：限价为何常常适得其反​

政府出于保护消费者或生产者的目的，经常会对市场价格进行管制。最常见的两种形式是价格上限（限价令）和价格下限（最低价格保护）。我们来分别分析它们的市场效果。

2.5.1 价格上限：好心可能办坏事​

价格上限（Price Ceiling）是政府规定的最高价格，不得超过这个价格销售。典型案例是房租管制和粮食价格管制。价格上限如果设置在均衡价格下方，会导致需求量大于供给量，形成短缺。

来看一个 Python 代码示例，用不超过三十行代码画出供需曲线，找到均衡点，并模拟一个价格上限管制下的效果：

import numpy as np

P = np.linspace(0, 10, 200)
def demand(p): return 10 - p
def supply(p): return -2 + p
QD = demand(P)
QS = supply(P)

# 均衡求解：10-P = -2+P => P = 6, Q = 4
eq_P = (10 + 2) / 2
eq_Q = 10 - eq_P

print("均衡价格: {:.1f}, 均衡数量: {:.1f}".format(eq_P, eq_Q))

ceiling = 3.5
shortage = demand(ceiling) - supply(ceiling)
print("限价 {} → 需求量 {:.1f}, 供给量 {:.1f}".format(
    ceiling, demand(ceiling), supply(ceiling)))
print("短缺量: {:.1f}".format(shortage))

运行这段代码，输出结果是：均衡价格 6.0，均衡数量 4.0。当政府设置价格上限为 3.5 时（在均衡价格 6.0 下方），需求量达到 6.5，供给量只有 1.5，短缺量高达 5.0。

价格管制的善意，不等于市场结果的善意。懂了这个道理，你就比大多数人对经济现象多了一层清醒。

价格上限在短期可能帮助特定群体（如低收入租户）获得更低的住房价格，但在中长期会引发一系列副作用：住房供给减少（开发商没有动力建更多房子）、质量下降（房东不愿投资维护）、黑市交易活跃（官方价格无法反映真实供需）。

2.5.2 价格下限：最低工资的代价​

价格下限（Price Floor）是政府规定的最低价格，不得低于这个价格销售。典型案例是最低工资法和农产品保护价。价格下限如果设置在均衡价格上方，会导致供给量大于需求量，形成过剩。

最低工资是一个特别值得深入讨论的例子。从经济学角度看，如果最低工资高于市场均衡工资水平，就会导致愿意工作的工人数量超过企业愿意雇佣的数量，从而产生结构性失业——不是因为经济衰退，而是因为工资被人为抬高之后，劳动力的供给超过了需求。当然，最低工资的具体影响还取决于劳动力市场的竞争程度、地区差异和执法力度等因素。

2.6 供需框架的应用​

这一章我们搭建了一个理解市场的基本框架：需求定律描述了买家对价格变化的反应，供给定律描述了卖家对价格变化的反应，两者的交叉点产生了均衡价格，均衡点随着供需曲线的移动而不断重塑，而价格管制在短期可能缓解某些群体的压力，但在中长期往往产生副作用甚至适得其反。

仔细想想，你身边的每一个交易行为背后，都有这套供需逻辑在默默运行。

再举一个有趣的例子：为什么同一款手机在新款发布前后的价格差异那么大？新款手机发布后，旧款机型往往大幅降价——这不是商家"良心发现"，而是供需机制在发挥作用。新款发布后，消费者对旧款的需求曲线急剧左移，因为"等等党"心理让很多人愿意等新款而不买旧款；同时旧款库存充足，供给曲线在短期内无法快速收缩。两相叠加，旧款机型只能在更低的价格水平上重新找到买主。

再看一些更宏观的。2021年到2022年期间，全球范围内出现了显著的通货膨胀，各国央行通过加息来应对。加息为什么能抑制通胀？用供需的框架来理解：加息提高了借贷成本，企业扩大生产的融资成本上升，供给曲线向左移动；同时，消费者因为利率上升而减少借贷消费，需求曲线也向左移动。两边同时向左移，但通常供给侧的收缩幅度更大，最终结果是价格水平下降，从而抑制通胀。

还有一个近年来很多人关心的领域：房价。限购政策、限售政策、限贷政策——这些政府对房地产市场的干预，本质上都是在需求曲线或供给曲线上做文章。限购提高了购房门槛，减少了需求曲线上的需求量；限售限制了二手房的流动性，减少了供给曲线上的供给量；限贷提高了购房资金成本，同时影响了需求和供给两侧。

用一句话来概括这一章的精髓：价格是市场信息的载体，均衡是市场运行的方向，供需关系的变动是所有经济波动的底层逻辑。掌握了这套语言，你就能在纷繁复杂的经济现象中看到本质。

当你点外卖时，平台根据供需状况动态调整配送费——雨天或者用餐高峰期，骑手供不应求，配送费就会上浮。当你用打车软件叫车时，系统显示"附近车辆较少，建议加价调度"，这个加价背后的逻辑同样是让价格反映供需紧张程度。当你在线上抢购某个限量商品时，"已售罄"这个状态本质上就是均衡价格为零、需求量远大于供给量的极端表现。

这套框架的厉害之处在于它的通用性。从微观层面的农产品价格、房租、外卖配送费，到宏观层面的通货膨胀、国际贸易、汇率，供需逻辑都是理解这些现象的第一把钥匙。当你下次看到一条"某某商品价格大涨"的经济新闻时，先问自己一个问题：这是需求曲线移动了，还是供给曲线移动了，还是两者同时移动了？

还有两个值得培养的习惯：第一，区分"短期"和"长期"。短期内的价格波动，往往是需求侧的变化主导；而长期的价格走势，则更多反映供给侧的调整。第二，养成"追问成本"的习惯。价格只是表面，成本才是底色。经济学不是教你精明算计，而是教你透过价格看到资源稀缺的真实成本。

下一章，我们要引入一个更精细的分析工具——弹性。需求价格弹性（Price Elasticity of Demand）衡量的是需求量对价格变化的敏感程度。为什么奢侈品降价未必带来销量大涨，而生活必需品涨价却让大家叫苦连天？第三章，我将用弹性的视角，把价格的传导机制讲得更精准。

如果你觉得这章内容有帮助，欢迎收藏慢慢消化。也欢迎把你在生活中观察到的供需现象写在评论区——比如最近哪个商品的价格变化让你觉得很有意思，或者你对某个价格现象有疑问，我们可以一起用今天学到的框架来分析。

## 课程介绍

经济学不是只有"钱"的学问，它是一套理解世界运行规律的思维方式。从"稀缺性"出发，一路讲到"国际贸易"，这套系列用通俗的语言、真实的案例和简短的 Python 示例，把微观经济学和宏观经济学最核心的东西讲清楚。

## 你将学到什么

| 模块 | 核心概念 |
|------|---------|
| 微观基础 | 稀缺性、机会成本、供需均衡、弹性 |
| 消费者理论 | 效用、边际效用、无差异曲线、消费者均衡 |
| 生产者理论 | 生产函数、成本曲线、利润最大化 |
| 市场结构 | 完全竞争、垄断、垄断竞争、寡头 |
| 劳动市场 | 人力资本、工资差异、失业类型、工会 |
| 宏观入门 | GDP、通胀、CPI、失业率 |
| 政策工具 | 货币政策、财政政策、乘数效应 |
| 国际贸易 | 比较优势、关税、汇率、全球化 |

## 学习建议

1. 每章独立成篇，可以按顺序阅读，也可以挑感兴趣的主题直接看
2. 章节末尾的 Python 示例建议动手跑一跑，改参数看结果变化
3. 结合生活案例思考：打车软件定价、电商促销、工资谈判……都是经济学
